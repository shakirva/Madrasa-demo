Technology & Design

Built with Modern Technology

A responsive, performant, and beautifully designed system built on cutting-edge frameworks

⚡

Next.js 16 & React 19

Server-side rendering, app router architecture, and Turbopack for blazing-fast development and production builds.

🔷

TypeScript

Full type safety across the entire codebase. Catch errors at compile-time with strict typing on all components and data.

🎨

Tailwind CSS

Utility-first styling with custom design tokens. Consistent emerald/teal theme with cream (#faf9f6) background.

✨

Framer Motion

Smooth, production-ready animations throughout — page transitions, card hovers, modal entries, and toast notifications.

📊

Recharts

Interactive data visualisation for attendance trends, fee collection analytics, and student performance bar charts.

🧠

Zustand State

Lightweight yet powerful global state for auth, language, and UI state. Persistent sessions with store hydration.

🇬🇧 English

Every label, button, heading, placeholder, notification, and toast message in the entire system is available in English. Automatic language switching with zero page reload.
